# Chat System Specifications

This directory contains specifications for the RAG Chatbot system.

## Files

- `chat-spec.md` - API specification document with endpoint details, request/response formats, and usage examples
- `chat-technical-spec.md` - Technical specification with architecture, data flow, security, performance, and deployment details

## Purpose

These specifications document the RAG (Retrieval Augmented Generation) Chatbot system that integrates humanoid robotics knowledge with AI-powered responses. The system uses OpenRouter for LLM responses and Qdrant for vector storage of documentation content.
